◆ pow() [3/3]

Tensor dl::pow ( Tensor &  base,
float  exponent 
)
noexcept

Computes the exponent -th power of each element in base and returns the resulting tensor.

This function is part of automatic differentiation. If base requires a gradient (dl::TensorImpl::requiresGrad() is true ), a reference to it is stored in the compute graph for the backwards pass. As such, do not delete the base tensor before the backwards pass finished. If you can't don't want to keep the tensor, move it into the compute graph via dl::pow(dl::Tensor&&,float).

dl::Tensor out;
{
dl::Tensor base = {1, 2, 3};
base->setRequiresGrad(true);
// The following will result in a segmentation fault since backward() is called when base was deleted
// out = dl::pow(base, 2);
// Solution: Move the tensor into the computation graph
out = dl::pow(std::move(base), 2);
}
out.backward();

Tensor dl::softmax ( Tensor &&  x)
noexcept

Computes the softmax function of the input vector.

Let \(x \in \mathbb R^n\) be a vector, the softmax is defined as

\[ \text{softmax}\colon \mathbb R^n \to \mathbb R^n ~\text{via}~ x\mapsto \left(\frac{e^{x_i}}{\sum_{j=1}^n e^{x_j}}\right)_{1\leq i\leq n}. \]
