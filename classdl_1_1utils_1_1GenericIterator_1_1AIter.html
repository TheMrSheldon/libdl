Detailed Description

template<typename T>
class dl::utils::GenericIterator< T >::AIter

The "abstract iterator" (AIter) provides a class interface for forward iterators (i.e. increment, dereference and equality operations) together with the option to make a copy and get a unique typeid to check if two iterators are of the same type.
