)
private

Called internally by the implementation for every chunk of data.

Writes the received data chunk to URLStreamBase::buffer, signals that new data is available such that a waiting (or a future call) to URLStreamBase::underflow() returns the fetched data. Afterwards it waits until this data was processed and returns.

Parameters
dataThe datachunk that was read.
userdataA pointer to some userdata (ignored).
Returns
true iff we want to continue reading data.
